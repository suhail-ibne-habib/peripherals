Bus

A bus is a communication system that transfers data between components inside a computer or between computers. The 8086 microprocessor uses address, data, and control buses.

Port

A port is a connection point or interface between a computer and an external or internal device. It can be used to connect devices like printers, modems, and hard drives.

Interfacing

Interfacing is the process of connecting the 8086 microprocessor with peripherals like memory and I/O devices to perform operations. It includes memory interfacing and I/O interfacing.
