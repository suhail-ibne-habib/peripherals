Internal Architecture of 8086

The Bus Interface Unit (BIU): Main function of BIU is divided into three sections.

  1. Calculating the 20-bit physical address.
  2. Provide the interface with memory and I/O devices.
  3. Provide pipelining by pre-fetching instructions and stored in the instruction queue.

❖ The BIU sends out addresses, fetches instructions from memory, reads data from ports and memory, and writes data to ports and memory.

❖ BIU handles all transfers of data and addresses on the buses for the execution unit.

Execution Unit (EU)

The Execution Unit (EU) is responsible for:

  1. Decoding and executing instructions provided by the BIU.
  2. Storing results in internal registers like AX, BX, CX, and DX.
  3. Updating the offset address of registers and status flags.

❖ The EU tells the BIU where to fetch instructions or data, decodes instructions, and executes them.

❖ It processes the instructions or data provided by the BIU and stores the result in the internal registers.

Bus Interface Unit (BIU)

The BIU handles all transfers of data and addresses on the buses for the execution unit. It sends out addresses, fetches instructions from memory, reads data from ports and memory, and writes data to ports and memory.

Interface

An interface is a shared boundary across which two or more separate components of a computer system exchange information. It can be between software, hardware, peripheral devices, or humans.

Pipelining

Pipelining is a technique where multiple instructions are overlapped in execution. The BIU uses pipelining to pre-fetch instructions and store them in the instruction queue for faster execution.

Execution Unit (EU)

The Execution Unit (EU) is responsible for decoding and executing instructions. It processes data provided by the BIU and stores results in internal registers like AX, BX, CX, and DX.

AX Register

The AX register is a 16-bit general-purpose register in the 8086 microprocessor. It is often used for arithmetic and data transfer operations.

BX Register

The BX register is a 16-bit general-purpose register in the 8086 microprocessor. It is often used as a base register for memory addressing.

CX Register

The CX register is a 16-bit general-purpose register in the 8086 microprocessor. It is often used as a counter in loop operations.

DX Register

The DX register is a 16-bit general-purpose register in the 8086 microprocessor. It is often used for I/O operations and storing high-order results of multiplication or division.
